## 📁 Featured Projects

### 👥 [Employee Risk Radar](https://github.com/MeetChhugani/employee-risk-radar)
An AI-powered employee attrition prediction and explanation system built to help HR departments identify at-risk staff, examine key turnover drivers, and deploy retention strategies.
* **Explainable AI**: Features local SHAP waterfall and global bar explanations demonstrating individual attrition triggers.
* **Performance**: Achieved a **ROC-AUC score of 0.77** utilizing a tuned **XGBoost Classifier** with **SMOTE** oversampling.
* **Generative HR Advisor**: Integrates LLaMA 3.3 70B via Groq to dynamically write tailored top-3 retention recommendations and run a chat assistant.
* *Tech Stack: Python, XGBoost, SMOTE, SHAP, Streamlit, Groq API (LLaMA 3.3).*

---

### 🧠 [QuizLab](https://github.com/MeetChhugani/QuizLab)
A premium AI-driven multiple-choice exam builder that parses textbooks or raw copy-pasted study guides to create high-fidelity assessments.
* **Adaptive Difficulty State Machine**: Scaled difficulty scaling (Easy $\leftrightarrow$ Medium $\leftrightarrow$ Hard) dynamically based on real-time user accuracy.
* **Adaptive Flowchart Pathway**: Custom horizontal glowing node flowchart pathway rendering on the scoreboard timeline showing user transitions.
* **Tech Stack & Integrity**: Dual fitz/pdfplumber document parser with text truncation safeguards, navigation locking, and Markdown/JSON/Anki TSV export suites.
* *Tech Stack: Python, Streamlit, Groq API (LLaMA 3.3), PyMuPDF.*
